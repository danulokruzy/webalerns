### ЭТАП 3️⃣: ЗАГРУЗИТЬ СКРИПТ В OBS

1. **Открыть OBS Studio**

2. **Перейти:** Tools → Scripts

3. **Левая панель:** Нажать "+" (Add Script)

4. **Выбрать файл:**
   ```
   c:\BotKruz\witcjet\tiktok-obs-plugin\obs-scripts\main_control.lua
   ```

5. **Нажать:** "Select" / "Открыть"

6. **В Properties установить:**
   - Server URL: `http://127.0.0.1:5000`
   - Auto Connect: ☑
   - Debug Mode: ☑

7. **Проверить логи:** должны быть зелёные ✓ сообщения

**✓ Завершено:** Lua скрипт загружен и готов

---

### ЭТАП 4️⃣: СОЗДАТЬ СЦЕНЫ

1. **OBS:** Tools → Scripts
2. **Найти скрипт** `main_control.lua` слева
3. **Нажать кнопку:** "Создать сцены"
4. **Ждать** завершения (конец логов)

**Результат:** Создадутся 4 сцены:
- ✓ Main Stream
- ✓ Intermission
- ✓ Donate Alert
- ✓ Dashboard

**✓ Завершено:** Все сцены готовы

## 🆘 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ

### Быстрые проверки

1. **Сервер запущен?**
   - Посмотреть на терминал
   - Если чёрный → Запустить start.bat

2. **OBS версия 27+?**
   - Help → About
   - Если ниже 27 → Обновить OBS

3. **Браузер показывает http://localhost:5000?**
   - Если нет → Сервер не запущен

4. **Скрипт видит сервер?**
   - OBS: Tools → Scripts
   - Если красные ошибки → Перезагрузить OBS

5. **Виджет всё ещё чёрный?**
   - Нажать F5 (Refresh в OBS)
   - Перезагрузить OBS полностью

### Если ничего не помогло

```bash
# 1. Остановить сервер (Ctrl+C в терминале)
# 2. Переустановить
cd c:\BotKruz\witcjet\tiktok-obs-plugin
install.bat

# 3. Запустить заново
start.bat

# 4. Перезагрузить OBS
# 5. Создать сцены заново
```
