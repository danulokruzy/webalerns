## 🎯 ЧТО ВНУТРИ?

✨ **5 готовых виджетов** с жидким стеклом эффектом:
- Progress Bar (таймер + счёт подарков)
- Circular Gauge (красивый gauge)
- Live Chat (чат с подарками)
- Donation Tracker (отслеживание донатов)
- VIP Alerts (уведомления о VIP)

🔧 **Для OBS Studio:**
- Lua скрипт (загружается в OBS)
- 4 готовые сцены (одной кнопкой)
- Browser Sources (все виджеты через браузер)

📡 **Backend (Python):**
- Flask сервер на localhost:5000
- REST API для управления
- WebSocket для real-time обновлений
- TikTok Live интеграция (опционально)

## 🚀 ГЛАВНЫЕ КОМАНДЫ

### 1️⃣ Первый раз (установка)
```bash
cd c:\BotKruz\witcjet\tiktok-obs-plugin
install.bat
```

### 2️⃣ Каждый раз (запуск)
```bash
start.bat
```
*(оставить открытым)*

### 3️⃣ В OBS
- Полуть **Tools** → **Scripts**
- Добавить: `obs-scripts/main_control.lua`
- Нажать: **"Создать сцены"**

**Done!** 🎉
