## Детальные шаги

### ЭТАП 0: Проверка требований ⏱️ ~2 минуты

#### Проверить Python
Открыть PowerShell и выполнить:
```powershell
python --version
```

✓ **Должно вывести:** `Python 3.x.x`

❌ **Если Python не найден:**
1. Скачать отсюда: https://www.python.org/downloads/
2. Установить с опцией **"Add Python to PATH"**
3. Перезагрузить PowerShell
4. Повторить команду

#### Проверить OBS Studio
- Открыть OBS
- Перейти: **Help** → **About**
- **Версия должна быть 27.0+**

❌ **Если версия ниже 27:**
1. Обновить OBS: https://obsproject.com/download
2. Параллельно старая версия удаляется

### ЭТАП 3: Загрузить Lua скрипт в OBS ⏱️ ~5 минут

#### Шаг 3.1: Открыть OBS
Запустить OBS Studio

#### Шаг 3.2: Перейти в Scripts
**OBS Menu:**
- Windows: **Tools** → **Scripts**
- Mac: **OBS** → **Tools** → **Scripts**

В левой панели будет список загруженных скриптов.

#### Шаг 3.3: Добавить скрипт
1. Нажать **"+"** (Add Script) в левой панели
2. Выбрать файл: 
   ```
   c:\BotKruz\witcjet\tiktok-obs-plugin\obs-scripts\main_control.lua
   ```
3. Нажать **"Select"** / **"Открыть"**

#### Шаг 3.4: Настроить свойства скрипта

В **Properties** (на панели справа) установить:

| Свойство | Значение | Описание |
|----------|----------|---------|
| **Server URL** | `http://127.0.0.1:5000` | URL backend сервера |
| **Auto Connect** | ☑ (включить) | Автоподключение при запуске |
| **Debug Mode** | ☑ (включить) | Для первого раза (потом можно выключить) |

#### Шаг 3.5: Проверить логи

В нижней части окна Scripts должны появиться зелёные сообщения:
```
[TikTok Plugin] Плагин загружен. Версия: 1.0
[TikTok Plugin] Сервер доступен: http://127.0.0.1:5000
```

✓ **Если видишь эти сообщения** → Всё подключилось!

❌ **Если видишь ошибки:**
- Проверить что сервер запущен (терминал открыт)
- Проверить URL сервера
- Перезагрузить OBS

### ЭТАП 4: Создание сцен ⏱️ ~2 минуты

Всё очень просто - **одна кнопка!**

#### Вариант A: Автоматическое создание (рекомендуется)

1. **OBS** → **Tools** → **Scripts**
2. Найти скрипт `main_control.lua` в списке слева
3. На панели справа нажать кнопку: **"Создать сцены"**

Скрипт автоматически создаст 4 сцены:

```
✓ Main Stream     → Главная сцена со скриптом
✓ Intermission    → Интермишн с gauge и чатом
✓ Donate Alert    → Алерты на донаты
✓ Dashboard       → Полная панель (4 виджета)
```

После успешного создания в консоли:
```
[TikTok Plugin] Создание сцен...
[TikTok Plugin] ✓ Сцены созданы успешно!
[TikTok Plugin] Создана сцена: Main Stream
[TikTok Plugin] Создана сцена: Intermission
...
```

#### Вариант B: Ручное создание (если нужно)

Если автоматическое не сработало - создать одну сцену вручную:

1. **OBS** → Правый клик на сцену → **New Scene**
2. **Name:** `Main Stream`
3. **Add Source** → **Browser**
4. Заполнить параметры:
   - **Name:** `Progress Widget`
   - **URL:** `http://localhost:5000/progress`
   - **Width:** `1920`
   - **Height:** `1080`
5. **OK**

Виджет загрузится и начнёт показывать прогресс.

### ЭТАП 5: Проверка и тестирование ⏱️ ~2 минуты

#### 5.1: Проверить что сцены созданы

**OBS** → Левая панель → **Scenes**

Должны видеть 4 сцены:
1. ✓ Main Stream
2. ✓ Intermission
3. ✓ Donate Alert
4. ✓ Dashboard

#### 5.2: Клик по сцене

Нажать на сцену `Main Stream`

**Результат:**
- Должна активироваться сцена
- В Preview должен показаться Progress виджет
- Таймер должен отсчитывать
- Данные должны обновляться

#### 5.3: Проверить другие виджеты

Нажимать на другие сцены:
- **Intermission** → gauge слева + чат справа
- **Dashboard** → все 4 виджета в сетке
- **Donate Alert** → трекер донатов

#### 5.4: Тестовый запрос к API

Открыть браузер и выполнить:
```
http://localhost:5000/api/widget/state
```

Должно вернуть JSON:
```json
{
  "rose_count": 0,
  "tiktok_count": 0,
  "timer_remaining": 120,
  "is_paused": false,
  "last_update": 1713398400
}
```

## FAQ

### Q: Где мои старые сцены?
**A:** Старые сцены не удаляются. Новые сцены добавляются рядом.

### Q: Можно ли изменить URL сервера?
**A:** Да! В OBS → Tools → Scripts → Properties → Server URL

### Q: Что если сервер выключится?
**A:** Виджеты перестанут обновляться. Перезапустить сервер (start.bat)

### Q: Как остановить/перезагрузить сервер?
**A:** В окне терминала нажать **CTRL+C** и снова запустить

### Q: Можно ли использовать на YouTube/Twitch?
**A:** Да, просто подключиться к нужной платформе в фронтенде

### Q: Что если браузер выводит "Cannot GET /progress"?
**A:** Сервер не запущен или URL неправильный. Проверить:
1. Сервер запущен (терминал открыт)
2. URL в браузере: http://localhost:5000/progress
3. Если не работает - перезагрузить сервер

### Q: Можно ли добавить свой виджет?
**A:** Да! Смотреть [docs/DEVELOPMENT.md](docs/DEVELOPMENT.md)
